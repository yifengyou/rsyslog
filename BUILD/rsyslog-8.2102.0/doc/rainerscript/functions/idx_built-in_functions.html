Built-in Functions¶

Following functions are available in Rainerscript.

  • ipv42num()
  • num2ipv4()
  • ltrim()
  • rtrim()
  • cnum()
  • cstr()
  • dyn_inc()
  • exec_template()
  • exists()
  • field()
  • format_time()
  • getenv()
  • int2hex()
  • is_time()
  • lookup()
  • parse_json()
  • parse_time()
  • previous_action_suspended()
  • prifilt()
  • random()
  • re_extract()
  • re_match()
  • replace()
  • script_error()
  • strlen()
  • substring()
  • tolower()
  • wrap()
